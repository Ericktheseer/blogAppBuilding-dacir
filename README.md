# Blog Application - Server

## App Description

This is the backend API server for a MEVN Blog Application. The server allows users to register, login, and manage blog posts.

## Features

- User registration
- User login
- Password hashing
- JWT authentication
- Create blog post
- View all blog posts
- View single blog post
- Update own blog post
- Delete own blog post
- Admin can delete any blog post
